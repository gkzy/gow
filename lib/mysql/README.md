# github.com/gkzy/gow/lib/mysql


### 使用方法：

*单数据库使用时*

```go
// 初始化链接
func init(){
    config:=&DBConfig{}
    mysql.InitDefaultDB(config)
}

// 按gorm方式使用
mysql.GetORM()
....
```

*多数据库时*

```go
// 初始化链接
func init() {
    // 配置多个DBConfig
    configs:=make([]*DBConfig,0)
    mysql.InitDB(configs)
}

//按gorm方式使用
db:=mysql.GetORMByName("user")
....
```
